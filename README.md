# Titanic

A machine learning project to predict passenger survival on the Titanic dataset.

## Project Structure

- `data/` - Dataset files
- `notebooks/` - Jupyter notebooks for exploration and modeling
- `src/` - Source code for data processing and modeling

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/titanic.git
    cd titanic
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the main script or explore the notebooks.

## Usage

- Modify and run notebooks in the `notebooks/` directory for EDA and modeling.
- Use scripts in `src/` for data processing and model training.

## Dataset

The Titanic dataset is available from [Kaggle](https://www.kaggle.com/c/titanic).
